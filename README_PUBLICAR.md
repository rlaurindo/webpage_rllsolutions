# RLL Solutions - Guia de publicacao

Este pacote esta pronto para publicar como site estatico. A pasta que deve ir para a plataforma e:

`outputs/rll-solutions-portfolio`

## Antes de divulgar

O site ja esta configurado com:

```text
WhatsApp: +351 965 695 323
Email: rllautomations@gmail.com
```

Se precisar trocar o numero:

1. Abra `script.js`.
2. Procure esta linha:

```js
const whatsappNumber = "351965695323";
```

3. Coloque o numero em formato internacional, sem espacos nem sinais.

Exemplo:

```js
const whatsappNumber = "351912345678";
```

## Publicar no Netlify

1. Aceda a `https://app.netlify.com/drop`.
2. Arraste a pasta `rll-solutions-portfolio` para a pagina.
3. O Netlify cria um link publico automaticamente.
4. Pode depois ligar um dominio proprio nas definicoes do site.

## Publicar no Vercel

1. Crie um projeto novo.
2. Importe ou carregue a pasta `rll-solutions-portfolio`.
3. Como e HTML estatico, nao precisa de build command.
4. Use a pasta raiz como output.

## Publicar no GitHub Pages

1. Crie um repositorio.
2. Envie os ficheiros desta pasta para o repositorio.
3. Em Settings > Pages, escolha a branch principal e a pasta raiz.
